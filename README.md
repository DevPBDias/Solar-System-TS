# Projeto App Trailers

**- Projeto Trybe Front-end**

A aplicação é um site que apresenta os planetas do sistema solar e missões da Nasa.

<details><summary>Stack utilizada</summary>
<ul>
  <li>Framework: React </li>
  <li>Linguagem: Javascript</li>
  <li>Estilização: Css3</li>
</ul>
</details>

<details><summary>Funcionalidades</summary>
<ul>
  
  * Utilizar JSX no React

  * Utilizar corretamente o método `render()` para renderizar seus componentes

  * Utilizar `import` para trazer componentes em diferentes arquivos

  * Criar componentes de classe em React

  * Criar múltiplos componentes a partir de um array

  * Fazer uso de `props` corretamente

  * Fazer uso de `PropTypes` para validar as `props de um componente`
</ul>
</details>

<details><summary>Aprendizados</summary>
 <ul>
  <li>Criar componentes react</li>
  <li>Criar cards</li>
  <li>Renderizar listas</li>
</ul>
</details>
